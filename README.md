# Koishi LiteX

本仓库提供了一个比 [Koishi Lite](https://koishi.chat/zh-CN/manual/starter/docker.html#%E5%90%AF%E5%8A%A8%E5%AE%B9%E5%99%A8) 还 lite 的 Koishi 实例, 不包含网页控制台, 仅用于 Entari 或 NoneBot 对接

## 预装适配器

- [`koishi-plugin-adapter-onebot`](https://github.com/koishijs/koishi-plugin-adapter-onebot)
- [`koishi-plugin-adapter-milky`](https://github.com/idranme/koishi-plugin-adapter-milky)
- [`@koishijs/plugin-adapter-satori`](https://koishi.chat/zh-CN/plugins/adapter/satori.html)
- [`@koishijs/plugin-adapter-qq`](https://koishi.chat/zh-CN/plugins/adapter/qq.html)
- [`@koishijs/plugin-adapter-discord`](https://koishi.chat/zh-CN/plugins/adapter/discord.html)
- [`@koishijs/plugin-adapter-telegram`](https://koishi.chat/zh-CN/plugins/adapter/telegram.html)
